## Facial Recognition Web App
This is a facial recognition web app that uses React, Typescript, and Faceapi. It communicates over api calls to gather facial data on the user to predict the gender and age along with a percent certainty. It uses an ML model to make these predictions. Make sure you allow the browser to access your webcam if you want to try it out. 

## How to run 
1. Clone the repository
2. Navigate to the project directory
3. Run `npm install` to install the dependencies
4. Run `npm start` to start the development server
